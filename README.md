Data Analysis for Toman Bike Share to address a central business question: 
Is it possible to increase prices next year?

The workflow covers five key stages:

1. Build a Database: creating a database, and importing raw data files.
   
2. Develop SQL Queries: Cleaning, joining tables, and calculating metrics like revenue and profit.
  
3. Connect Power BI: Establishing a connection between your SQL database and Power BI.
  
4. Build a Dashboard: Designing an interactive report with visuals, including hourly revenue analysis, trends, and seasonal data.
   
5. Make Recommendations: Analyzing the data to provide an informed, conservative recommendation for a price increase.
